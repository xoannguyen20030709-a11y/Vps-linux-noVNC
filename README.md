# 🚀 Linux VPS noVNC on Railway

## ✨ Features
- Ubuntu 22.04
- XFCE Desktop (lightweight)
- noVNC Web Access
- Firefox Browser
- Audio support
- File Manager
- Optimized for 2vCPU / 2GB RAM

---

## 🔑 Login
- VNC Password: `railway`

---

## 🚀 Deploy to Railway

### Step 1: Fork repo
Fork repo này về GitHub của bạn

### Step 2: Railway
- Vào https://railway.app
- New Project → Deploy from GitHub Repo
- Chọn repo này

### Step 3: Đợi build xong
Railway sẽ tự build Docker

### Step 4: Truy cập
Mở domain Railway cấp  
👉 `https://<app>.up.railway.app`
